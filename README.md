# ScriptLynx Chrome Plugin:
Simple plugin to block resources on any website based on patterns or paths.
Use this, to combat methods like "console-ban" and other nasty scripts effectively.

# How to install & use:
- Clone repo
- Open chrome/chromium/brave extension manager with developer options enabled
- Click "Load unpacked" and select repo folder
- Install done!
- Now rightclick plugin icon and choose "options" to visit config page
- There add entrys as descriped and hit save :)
